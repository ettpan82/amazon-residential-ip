## 住宅IP、双ISP、机房IP——傻傻分不清楚？

市面上有几个概念经常被混用，这里快速理清一下。

**机房 IP（Datacenter IP）**：IP 归属是某个数据中心，不是住宅运营商。可以很便宜，但对亚马逊、TikTok 这类平台来说，识别度极高，是最容易被风控的类型。

**双 ISP（Dual ISP）**：同时显示两个 ISP 来源，有时是为了看起来更像住宅 IP。但注意，双 ISP 不等于真实住宅 IP。很多服务商靠这个标签卖贵价。

**真实住宅 IP（Genuine Residential IP）**：由美国本地运营商（如 AT&T、Comcast）直接分配给真实居民家庭使用的地址。真住宅宽带一定是双 ISP，但双 ISP 不一定是真住宅。

一句话总结：**真实住宅 IP 天然包含双 ISP 特征，但双 ISP 标签本身不等于来自真实住宅宽带**。

这个区别，对亚马逊账号稳定性影响很大。
